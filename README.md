# sc-simple-demo
Substrate Connect Simple demo

## Getting Started

1. Clone the whole repository.

```bash
git clone https://github.com/wirednkod/sc-simple-demo/
```

2. Install all dependencies

```bash
yarn install
```

3. Compile all project

```bash
yarn build
```

4. Run the code

```bash
yarn run dev
```
